## Introduction

Welcome to the **Mosaic Global Cuisine Restaurant** web platform—a premium digital experience crafted to mirror the elegance and world-class hospitality of our physical dining space. Built with modern web technologies (React & Tailwind CSS), this landing page is designed to offer our guests an immersive preview of what awaits them inside Hotel Country Inn, Hebbal. 

From exploring our extensive, freshly prepared buffet spreads to easily booking a table for the evening, the Mosaic web application ensures an uncompromisingly smooth and visually stunning user experience before a guest even steps foot through our doors.

## About Mosaic

## Features of this Project
- **Modern UI/UX:** Built with a sophisticated, premium feel using Framer Motion for subtle scroll animations.
- **Responsive Layout:** Beautifully scaled interfaces from mobile phones to ultra-wide desktop monitors.
- **Interactive Menu:** Browse through dynamic menu sections ranging from Breakfast Buffets to A La Carte and Desserts.
- **Direct Reservations:** Integrated, user-friendly reservation forms and direct phone linkages.

## Getting Started

To run this project locally, simply clone the repository and run the following commands:

```bash
# Install dependencies
npm install

# Run the local development server
npm run dev
```

Your local server will start, generally accessible at `http://localhost:5173/`.
